# HR-DASHBOARD-tableau-

**HR Analytics Dashboard - Tableau Project**
This project is an end-to-end HR Analytics solution developed in Tableau. It transforms raw employee data into an interactive, professional-grade dashboard designed to help HR managers track hiring trends, employee retention, and workforce demographics.

**üöÄ Project Overview**
The dashboard provides a high-level summary of human resources KPIs and a detailed employee list, utilizing a custom-designed background for a modern, high-end look.

**Key Metrics Tracked (KPIs)**

Total Hired: Total count of employees ever hired.

Active Employees: Current workforce strength (excluding terminated staff).

Terminated Employees: Total number of employees who have left the company.

Hiring Trends: Analysis of recruitment over time via line charts.

**üìä Dashboard Features**

Dynamic Overview: Visualizes total headcount and termination rates.

Geographic Distribution: A map view showing employee distribution across different states.

Demographic Insights: Breakdown of the workforce by Gender, Age, and Department.

Interactive Filters: Users can filter data by Date, Department, and Job Title.

Custom UI: Includes rounded edges, shadows, and a clean navigation bar using custom-designed background images.

**üìÇ Project Structure**

Data Connection: Connected to a single CSV/Excel file containing employee records.

Calculated Fields: Custom logic created for "Active" vs. "Terminated" status.

Worksheets: Individual sheets for Maps, Line Charts, and BANs (Big Aggregate Numbers).

Final Dashboard: A packaged Tableau Workbook (.twbx) with a floating background layout.
